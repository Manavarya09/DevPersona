# DevPersona

DevPersona transforms your GitHub profile into a stunning portfolio, resume, and brand identity. Built with React + Tailwind CSS.

## Features

- Instant portfolio generation from GitHub username
- AI-enhanced project descriptions
- Multiple identity-based themes
- Resume generation and export
- Editable dashboard

## Tech Stack

- React + Vite
- Tailwind CSS
- Node.js + Express

## Getting Started

```bash
npm install
npm run dev
```
